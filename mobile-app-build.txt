# Mobile App Build Instructions
# ===========================

# 1. Install Expo CLI globally (if not already installed)
npm i -g expo-cli

# 2. Navigate to the mobile app directory
cd mobile-app

# 3. Install all dependencies
npm install

# 4. Start the development server
npm start

# Alternative commands for specific platforms:
# npm run android  # Run on Android emulator/device
# npm run ios      # Run on iOS simulator/device  
# npm run web      # Run on web browser

# Prerequisites:
# - Node.js installed
# - For iOS: Xcode and iOS Simulator
# - For Android: Android Studio and Android emulator
# - For web: Any modern browser

# Troubleshooting:
# - If you get "expo command not found", try: npx expo start
# - For Android, ensure Android SDK is properly configured
# - For iOS, ensure Xcode command line tools are installed
